Linear Gradient Generator.

Description

This application randomly generates two linear gradients, and also allows for the user to manually create their own linear gradient using a color picker. Previously, I've found it difficult to settle on colors to use for linear gradients, so I think that randomly generating them and then seeing what looks good is at least a solution for the issue.
